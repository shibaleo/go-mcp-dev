# PSN-1: 個人用ペルソナ

## 概要

プライベート・趣味・PKMなどの個人的な活動を行う際のペルソナ。
日常生活の効率化や個人的な情報管理、学習・クリエイティブ作業の最適化が主な関心事。

---

## ペルソナ概要

| 属性カテゴリ | 詳細 |
|-------------|------|
| **ID** | PSN-1 |
| **名前** | 個人用ペルソナ |
| **利用文脈** | プライベート、趣味、PKM、日常タスク、個人学習・創作活動 |
| **使うサービス** | Notion個人用、Google Calendar、Microsoft Todo、TickTick、Todoist、GoodNotes、Kindle、Zaim、Toggl Track、Claude、Fitbit、Tanita HealthPlanet、Suica、銀行アプリ、Dropbox、Google Drive、Mega |
| **技術スキル** | Webアプリ利用、基本的なLLM操作 |
| **行動特性** | シンプルさ重視、即時性重視、思考と行動の統合を求める |

---

## 課題（Pain）

1. **思考・タスク・情報の分散**
   - タスクやアイデア、情報が頭の中や複数アプリに散らばる
   - 思い出すことに脳リソースを割かれて創造的思考に集中できない

2. **サービス間データ統合の手間**
   - 複数アプリのデータをまとめる作業が手間
   - 手書きノートやPDFの情報が構造化されていない

3. **タスク実行の非効率**
   - 簡単なタスクをLLMが直列に実行するため時間がかかる
   - 認証やツール実行の確認が頻繁に必要でUXが低下

4. **情報の混在による集中妨害**
   - 個人用の思考・趣味作業中に仕事や開発情報が混ざるとイライラ
   - 完全隔離は手間なので、思考優先での柔軟なフィルタリングが必要

5. **学習・知識最適化の難しさ**
   - LLMが広範な知識を拾うが、自分の理解度や信頼性と合わない
   - 自分のメモや教科書範囲に絞ったコンテキスト生成ができない

6. **自己管理・生活情報の最適化**
   - 健康・睡眠・財務・人間関係の情報を含めて適切に管理・通知されない

---

## 得たいもの（Gain）

1. **頭のすっきりと集中環境**
   - 思考に必要な情報だけが整理され、創造的作業に集中できる

2. **タスク登録・整理の自動化**
   - アプリUIを開かずに自動でタスク登録され、必要なときだけ確認可能
   - 関連性のない軽量タスクは並列実行で効率化

3. **最小限のツール操作**
   - LLMが必要なツールだけを選択・実行し、ユーザーは操作を意識しない
   - 失敗時のみ通知される

4. **コンテキスト管理**
   - 個人用の思考・PKM情報を優先的に扱い、不要な仕事・開発情報は後回し
   - 情報の多少の混在は許容

5. **学習支援**
   - 自分のメモや読んでいる教科書に検索範囲を絞り、LLMにコンテキスト生成させる
   - 過去の自分の知識＋少し背伸びした知識にアクセスできる

6. **俯瞰的自己分析**
   - 行動・思考・タスクを自動で記録・整理
   - いつでも振り返り・分析できる

7. **生活・健康サポート**
   - 健康・睡眠・財務・人間関係も適度にモニター
   - 頑張りすぎや偏りを検知して注意喚起してほしい

8. **クリエイティブな即時行動**
   - アイデアをすぐに形にできる（プログラム実装や数式の定式化など）
   - 思考→実行の流れをスムーズに
