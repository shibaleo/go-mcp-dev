# PSN-2: 仕事用ペルソナ

## 概要

経理業務・税理士補助業務を行う際のペルソナ。  
大量の単純ルーティン作業を自動化し、人間は「判断と責任」に集中することを目的とする。  
業務フローや組織を変えることは目指さず、**個人の裁量範囲内で効率を最大化**する。

## 使用サービス・ツール

### 業務必須
- Excel
- Outlook
- Teams
- Microsoft Todo
- 勘定奉行クラウド

### 個人最適化用途
- Notion（業務メモ・整理）
- ローカルPC（最重要・原則ここで完結）

---

## 課題（Pain）

1. **紙中心・手作業中心の業務**
   - 紙帳票 → Excel の転記
   - Excel → Excel のコピー＆整形
   - 単純だが量が多く、精神的に消耗する

2. **時間が奪われる構造**
   - 創造性や判断力は不要だが、時間だけが消費される
   - 税理士試験の勉強時間を確保できない

3. **業務構造の不可視性**
   - タスクの依存関係や並行可能性が見えない
   - プロジェクト管理ツールは組織的に使えない

4. **セキュリティ制約**
   - 業務データをクラウドに送ると即アウト
   - 原則ローカル完結が必須

---

## 期待・得たいもの（Gain）

1. **大量ルーティンの完全自動化**
   - OCR / AI-OCR
   - データ整形・転記・DB化
   - バックグラウンド・並列処理

2. **判断に集中できる環境**
   - 人間は「正しいか」「責任を負えるか」だけを見る
   - 作業はLLM・自動化に任せる

3. **業務の構造可視化**
   - タスクをDAGとして表示
   - 依存関係・実行順・並列性が一目で分かる

4. **時間の再投資**
   - 空いた時間を税理士試験・専門知識に充てる

---

## LLMとの責任境界（最重要）

### 完全自動でOK（責任は人間が負う）
- OCR / AI-OCR
- PDF → Excel
- Excel → Excel の整形・転記
- 大量ルーティンの並列処理
- データベース化・構造化
- 下準備データの生成
- タスク構造（DAG）の生成

### 補助・候補提示までOK
- 仕訳案の生成
- チェック観点の列挙
- 説明文・資料・メールの下書き
- 税務論点の洗い出し（結論は出さない）

### 絶対に人間が判断・実行
- 顧客・上司・同僚へのコミュニケーション
- 税務判断の確定
- 対外的な説明責任を伴う行為
- 「正しい」と断定する行為

> **理由**  
> データの誤りは修復できるが、人間関係は修復できないため。

---

## 行動特性

- 業務フローは変えない
- 組織改革はしない
- 個人の範囲で最適化する
- 責任は自分で引き受ける
- 無駄な待ち時間・直列処理に強いストレスを感じる

---

## このペルソナの本質（One-liner）

> **「LLMに作業を任せ、自分は判断と責任だけを引き受ける税務人材」**
